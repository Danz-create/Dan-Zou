## Overview

A planning tool that recommends optimal deployment locations for **Automated External Defibrillators (AEDs)** in dense urban environments, using fine-grained building distribution and population density data. Implemented as a registered software system (Version 1.0).

## Highlights

- Ingests urban building footprints and demographic layers
- Runs spatial optimization to maximize population coverage and minimize median response time
- Interactive map-based interface for planners to preview and tweak deployments
